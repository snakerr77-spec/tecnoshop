# TecnoShop Cliente — React

Site público da TecnoShop desenvolvido com React, TypeScript, Vite e Framer Motion.

## Executar no computador

```bash
npm install
npm run dev
```

## Gerar versão de produção

```bash
npm run build
```

A pasta `dist` contém a versão pronta para publicação. O projeto usa `base: "./"`, permitindo publicar em GitHub Pages, Cloudflare Pages ou outra hospedagem estática.

## Principais arquivos

- `src/App.tsx`: componentes, catálogo e interações.
- `src/styles.css`: visual, animações e responsividade.
- `src/products.ts`: produtos demonstrativos.
- `public/assets`: logo e imagem principal.
